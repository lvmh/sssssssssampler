HOW TO USE:

1. Copy libsssssssssampler.dylib to:
   ~/Library/Audio/Plug-Ins/VST3/

2. Restart your DAW (Logic Pro, Ableton, Studio One, etc.)

3. Add plugin to a track

4. Open the editor window

5. Adjust parameters:
   - Sample Rate: 1,000 - 96,000 Hz
   - Bit Depth: 1 - 24 bits
   - Jitter: 0 - 100%
   - Filter: 0 - 100%
   - Mix: 0 - 100%

6. Watch the ASCII grid respond to audio in real-time
