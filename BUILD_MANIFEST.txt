HOW TO USE
──────────

1. Launch Ableton Live (or your DAW)

2. Rescan plugins
   - Ableton: Options → Plug-in Manager → Rescan
   - Or force rescan by deleting cache

3. Add sssssssssampler to a track
   - Both VST3 and CLAP formats supported
   - Works as an audio effect (VST3/CLAP plugin)

4. Route audio through the plugin
   - The visual output reacts to:
     • Sample rate reduction (target_sr parameter)
     • Bit depth crushing (bit_depth parameter)
     • Dry/wet mix ratio (mix parameter)
     • Jitter/timing (jitter parameter)
     • Filter cutoff + pole count (filter parameters)

5. Observe real-time visual reactivity
   - Audio amplitude drives layer switching (1–5 overlays)
   - Sample rate degradation creates visual artifacts
   - Motion system keeps screen active (never static)
   - Colors adapt to selected theme
